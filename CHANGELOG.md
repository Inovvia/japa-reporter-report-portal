# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-05

### Added

- Initial release
- Full integration with Japa test runner v5+
- Support for Report Portal API v2
- Suite and group hierarchy mapping
- Test status mapping (passed, failed, skipped, todo)
- Error logging with stack traces
- API key authentication
- OAuth authentication support
- Launch attributes and descriptions
- Debug mode for troubleshooting
- Launch merge support for parallel runs
- TypeScript type definitions
- Comprehensive documentation

### Features

- `RPReporter` class for easy integration
- Configurable via environment variables
- Code reference tracking for test navigation
- Automatic error description extension
- Support for custom HTTP headers
- Axios configuration passthrough
